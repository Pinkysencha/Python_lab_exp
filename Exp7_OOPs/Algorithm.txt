### single Inheritance

1. Start
2. Define parent class `Animal` with method `sound()`
3. Define child class `Dog` inheriting `Animal`
4. Create object of `Dog`
5. Call inherited method `sound()`
6. Display output
7. Stop

###Multiple Inheritance

3. Define class `Skills` with method `run()`

7. Display output

### Multilevel Inheritance

1. Start
2. Define class `Grandparent` with method `house()`
3. Define class `Parent` inheriting `Grandparent`
4. Define class `Child` inheriting `Parent`
5. Create object of `Child`
6. Call `house()`, `car()`, and `bike()`
7. Display output
8. Stop



### Hierarchical Inheritance

1. Start
2. Define class `Person` with method `skills()`
3. Define class `Student` inheriting `Person`
4. Define class `Employee` inheriting `Person`
5. Create objects of `Student` and `Employee`
6. Call `skills()` using both objects
7. Display output
8. Stop
